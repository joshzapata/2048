lunar2048

A small clone of 1024

View on GitHub Download .zip Download .tar.gz

HOW TO PLAY: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

NOTE: The game on this site is the original version of 2048 created by Gabriele Cirulli. Based on 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer. Modified by joshzapata.
